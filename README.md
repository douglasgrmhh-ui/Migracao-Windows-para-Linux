# 🐧 Migração do Windows para o Linux Mint

## 📖 Sobre o projeto

Este repositório documenta minha experiência de migração do Windows para o Linux Mint como sistema operacional principal.

O objetivo foi registrar as dificuldades encontradas, as ferramentas instaladas, os comandos aprendidos e a evolução no uso do terminal, Git, GitHub, Python e Visual Studio Code.

Este projeto faz parte do meu portfólio em Tecnologia da Informação e representa uma etapa importante no desenvolvimento da minha autonomia técnica.

## 🎯 Objetivos

- Aprender a utilizar Linux no dia a dia.
- Desenvolver familiaridade com o terminal.
- Instalar e configurar ferramentas de desenvolvimento.
- Aprender comandos básicos do sistema.
- Resolver problemas de software e hardware.
- Utilizar Git e GitHub para versionamento.
- Documentar minha evolução técnica.

## 🧰 Ferramentas utilizadas

- Linux Mint
- Terminal Bash
- Git
- GitHub
- Visual Studio Code
- Python
- APT
- Markdown
