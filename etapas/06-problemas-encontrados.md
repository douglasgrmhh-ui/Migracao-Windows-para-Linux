# 06 - Problemas encontrados

Durante a migração encontrei diversos desafios que exigiram pesquisa, testes e paciência.

Entre eles:

- Configuração inicial do sistema.
- Adaptação ao terminal.
- Instalação de programas.
- Configuração do Git.
- Publicação dos primeiros projetos no GitHub.
- Problemas com dispositivos, como o Bluetooth.
- Diferenças entre Windows e Linux.

Cada problema resolvido aumentou minha familiaridade com o sistema e contribuiu para desenvolver autonomia na resolução de dificuldades técnicas.
