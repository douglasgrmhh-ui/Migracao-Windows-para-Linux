# 04 - Configuração do Git e GitHub

Após me familiarizar com o Linux, comecei a aprender Git e GitHub para controlar versões dos meus projetos.

Durante esse processo aprendi a:

- Instalar o Git.
- Configurar usuário e e-mail.
- Inicializar repositórios.
- Criar commits.
- Organizar projetos.
- Publicar repositórios no GitHub.
- Utilizar o VS Code integrado ao Git.

O versionamento se tornou parte fundamental do meu fluxo de estudos, permitindo registrar cada evolução dos projetos e acompanhar meu aprendizado de forma organizada.
