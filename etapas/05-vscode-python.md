# 05 - Visual Studio Code e Python

Com o ambiente Linux configurado, instalei o Visual Studio Code e preparei o sistema para estudar programação em Python.

Durante essa etapa aprendi a:

- Instalar o Visual Studio Code.
- Configurar extensões.
- Utilizar o terminal integrado.
- Criar meus primeiros projetos em Python.
- Organizar arquivos utilizando boas práticas.

A integração entre Linux, Git e VS Code tornou o ambiente de desenvolvimento muito mais produtivo para os estudos.
